# Een gegevensfeed-taak opnieuw uitvoeren

De interface voor gegevensinvoer in Adobe Analytics maakt gegevensdoorvoerbeheer veel eenvoudiger. Als u toegang hebt om gegevensfeeds te beheren, kunt u een gegevensfeed-taak op zichzelf uitvoeren in de interface voor gegevensinvoer, zonder dat het team voor klantenondersteuning de gegevensdoorvoerbestanden opnieuw hoeft te verzenden.
