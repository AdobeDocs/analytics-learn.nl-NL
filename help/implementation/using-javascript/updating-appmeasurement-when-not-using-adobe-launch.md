# AppMeasurement bijwerken wanneer [!DNL Experience Platform Launch] niet wordt gebruikt {#updating-appmeasurement-when-not-using-adobe-launch}

In deze video ziet u hoe u de code appMeturement kunt bijwerken wanneer u Adobe [!DNL Experience Platform Launch] niet gebruikt. Dit omvat waar u de code wilt ophalen en waar u deze wilt bijwerken.

>[!VIDEO](https://video.tv.adobe.com/v/25913/?quality=12&learn=on)

>[!NOTE]
>
>Als u een niet-Adobe-oplossing voor tagbeheer gebruikt, moet u [!DNL AppMeasurement] -updates behandelen alsof u geen oplossing voor tagbeheer gebruikt. Betekenis, waar u uw code van AppMeasurement hebt, dat is waar u de update (wint de nieuwe code terug en vervang de oude code in het dossier of in TMS) uitvoert. Als u [!DNL Experience Platform Launch] gebruikt, worden updates snel en gemakkelijk en hoeft er geen code te worden gekopieerd of geplakt.
